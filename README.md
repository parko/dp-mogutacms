# dp-mogutacms
Payment module for MogutaCMS

# Инструкция

Для установки либо воспользуйтесь стандартной формой установки плагинов, предоставляемой Moguta CMS (для этого в панели управления перейдите на вкладку плагины, затем нажмите "Загрузить плагин" и выберите zip архив); либо распакуйте плагин в папку mg-plugins. После этого активируйте плагин.


Для того, чтобы покупатель мог выбрать delta processing как метод оплаты, следует включить его в доставку. Для этого в панели управления зайдите на вкладку Настройки - Доставка и нажмите на "Редактировать" любого из способов. В списке "Варианты оплаты" следует поставить галку напротив DeltaProcessing.


Для того, чтобы покупатель смог оплатить заказ через delta processing, следует ввести id магазина и секретный ключ в настройках. Для этого в панели управления перейдите на Настройки - Оплата и нажмите на "Редактировать" напротив DeltaProcessing.

# Информация для разработчиков/верстальщиков.

По умолчанию плагин ориентируется на стандартный шаблон Moguta CMS, а именно: ищет div с классом "main-block" и добавляет в него элемент (кнопку оплаты). Если в вашем шаблоне другая система наименования, то следует добавить shotcut в месте, где вы вызываете код с формированием ссылок на оплату. Для этого вставьте в html строку "[delta-payment]".
